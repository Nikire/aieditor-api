## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Prisma DB Migrations

```bash
# run migrations
yarn run prisma migrate dev

# create new migration
yarn run prisma migrate dev --name <your_migration_name>
```
